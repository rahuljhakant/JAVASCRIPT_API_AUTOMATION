# Advanced Web Application Security

## Overview

This directory contains advanced web application security testing techniques including API security and GraphQL security.

## Tutorials

### 1. Advanced API Security
**File:** `api-security.mjs`

Advanced API security testing including authentication, authorization, rate limiting, and API-specific vulnerabilities.

### 2. GraphQL Security
**File:** `graphql-security.mjs`

GraphQL-specific security testing including introspection queries, query depth attacks, and authorization bypass.

## Key Topics

- REST API security
- GraphQL security
- API authentication and authorization
- Rate limiting and throttling
- API versioning security
