# Enterprise Penetration Testing

## Overview

This directory contains tutorials on enterprise-scale penetration testing and social engineering testing.

## Tutorials

### 1. Enterprise Penetration Testing
**File:** `enterprise-pentest.mjs`

Large-scale penetration testing methodologies, planning, and execution for enterprise environments.

### 2. Social Engineering Testing
**File:** `social-engineering.mjs`

Social engineering testing techniques including phishing, pretexting, baiting, and tailgating.

## Enterprise Testing Considerations

- Multi-site testing
- Large-scale network testing
- Compliance requirements
- Stakeholder communication
- Risk management

## ⚠️ Legal and Ethical Guidelines

- Always obtain proper authorization for enterprise testing
- Follow scope limitations strictly
- Respect privacy and data protection
- Document all activities thoroughly
