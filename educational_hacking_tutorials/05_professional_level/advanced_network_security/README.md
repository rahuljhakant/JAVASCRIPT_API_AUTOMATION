# Advanced Network Security

## Overview

This directory contains advanced network security testing techniques including network penetration and wireless security.

## Tutorials

### 1. Network Penetration
**File:** `network-penetration.mjs`

Network penetration techniques, network segmentation testing, and VLAN hopping.

### 2. Wireless Security
**File:** `wireless-security.mjs`

Wireless security testing, WPA2/WPA3 vulnerabilities, and wireless protocol analysis.

## ⚠️ Legal and Ethical Guidelines

- Only test networks you own or have explicit written permission to test
- Never access wireless networks without authorization
- Follow responsible disclosure practices
