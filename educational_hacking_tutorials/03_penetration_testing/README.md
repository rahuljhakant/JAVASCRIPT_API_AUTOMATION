# Penetration Testing Tutorials

## Overview

This directory contains comprehensive penetration testing methodologies, from reconnaissance through exploitation to professional reporting.

## Tutorials

### 1. Reconnaissance
**File:** `reconnaissance.mjs`

Learn information gathering techniques, DNS enumeration, and reconnaissance methodologies.

### 2. Vulnerability Scanning
**File:** `vulnerability-scanning.mjs`

Automated vulnerability discovery and security header analysis.

### 3. Exploitation Techniques
**File:** `03_exploitation.mjs`

Safe exploitation methods, proof of concept development, and exploit chain building.

### 4. Post-Exploitation
**File:** `04_post_exploitation.mjs`

Persistence mechanisms, lateral movement, data exfiltration, and privilege maintenance.

### 5. Security Reporting
**File:** `05_security_reporting.mjs`

Professional security report generation, risk assessment, and remediation recommendations.

## ⚠️ Legal and Ethical Guidelines

- Only perform penetration testing on systems you own or have explicit written authorization
- Follow responsible disclosure practices
- Document all testing activities
- Respect scope limitations
