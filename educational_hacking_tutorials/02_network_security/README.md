# Network Security Tutorials

## Overview

This directory contains tutorials on network security fundamentals, including port scanning, protocol analysis, traffic analysis, firewall testing, and IDS/IPS testing.

## Tutorials

### 1. Port Scanning
**File:** `port-scanning.mjs`

Learn network reconnaissance techniques and port scanning methods for security testing.

### 2. Protocol Analysis
**File:** `02_protocol_analysis.mjs`

Analyze HTTP/HTTPS, WebSocket, DNS, SMTP, and FTP protocols for security vulnerabilities.

### 3. Traffic Analysis
**File:** `03_traffic_analysis.mjs`

Learn network traffic analysis, packet analysis, and network forensics techniques.

### 4. Firewall Testing
**File:** `04_firewall_testing.mjs`

Test firewall rules, port filtering, and firewall bypass techniques.

### 5. IDS/IPS Testing
**File:** `05_ids_ips_testing.mjs`

Test intrusion detection and prevention systems, including evasion techniques.

## ⚠️ Legal and Ethical Guidelines

- Only test systems you own or have explicit written permission to test
- Never scan or test systems without authorization
- Follow responsible disclosure practices
- Respect privacy and data protection laws
